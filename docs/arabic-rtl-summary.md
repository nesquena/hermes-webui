# إضافة اللغة العربية ودعم RTL إلى Hermes WebUI

## نظرة عامة
تمت إضافة دعم كامل للغة العربية إلى واجهة Hermes WebUI، مع ترجمة شاملة واتجاه نص من اليمين لليسار (RTL) وخط عربي احترافي (ثمانية Thmanyah Sans).

## ما تم إضافته

### 1. الترجمة العربية (600+ مفتاح)
- **الواجهة العامة**: التنقل، التبويبات، الأزرار، القوائم
- **الإعدادات**: المظهر، التفضيلات، المزودون، الإضافات، النظام، كلمة المرور
- **المحادثات**: إرسال، تعديل، نسخ، إعادة توليد، الموافقات، التوضيحات
- **المساحات**: إدارة الملفات، رفع، حذف، إعادة تسمية
- **المهام المجدولة (Cron)**: إنشاء، تعديل، حذف، جدولة، إشعارات
- **كانبان**: المهام، الحالات، المسند إليهم، التوزيع
- **المهارات**: إنشاء، تعديل، حذف، الفئات
- **الملفات الشخصية**: إنشاء، تبديل، حذف، المفاتيح
- **السجلات والإحصائيات**: التصفية، المستويات، العرض
- **لوحة الإعداد الأولي (Onboarding)**: جميع الخطوات والرسائل

### 2. خط ثمانية (Thmanyah Sans)
- 5 أوزان: Light (300)، Regular (400)، Medium (500)، Bold (700)، Black (900)
- تنسيق woff2 المضغوط (حجم إجمالي ~385KB)
- تطبيق تلقائي عند اختيار العربية مع الحفاظ على الخطوط أحادية المسافة للأكواد

### 3. نظام RTL التلقائي
- عند اختيار العربية من Settings → Language، يُفعّل RTL فوراً
- يحترم التفضيلات اليدوية (إذا عطّل المستخدم RTL يدوياً، لا يُعاد تفعيله)
- دعم للغات RTL الأخرى: الفارسية، العبرية، الأردية، الكردية...

## تفاصيل تغييرات RTL في CSS

### تخطيط الصفحة
- `flex-direction: row-reverse` على `.layout` لنقل الشريط الجانبي لليمين
- عكس الحدود (`border-right` ↔ `border-left`) للشريط الجانبي ولوحة الملفات
- مؤشر التبويب النشط في شريط التنقل على اليمين
- مقبض تغيير حجم الشريط الجانبي على اليسار

### الجوال (max-width: 640px)
- الشريط الجانبي ينزلق من اليمين بدل اليسار
- حركة الطي معكوسة (`translateX(14px)` بدل `translateX(-14px)`)

### قائمة المحادثات الجانبية
- padding-right للتحويم يتحول إلى padding-left
- مؤشرات التنبيه وقوائم الإجراءات على اليسار
- وقت الجلسة ينتقل لليسار (`margin-right: auto`)
- قائمة الإجراءات المنبثقة تفتح من اليمين

### المحادثات
- `padding-right: 30px` بدل `padding-left` للرسائل
- رسائل المستخدم تميل يمين (`margin-right: auto`)
- رسائل المساعد تميل يسار (`margin-left: auto`)

### عناصر المحادثة
- هوامش الأدوات والتفكير والملفات تنعكس (`margin-left` ↔ `margin-right`)
- القوائم: تعداد نقطي/رقمي على اليمين
- الاقتباسات: الحدود على اليمين

### نص مختلط (عربي + إنجليزي)
- `unicode-bidi: plaintext` على الفقرات والعناوين للكشف التلقائي عن اتجاه النص
- الروابط معزولة (`unicode-bidi: isolate`) لمنع تشوهها
- الأكواد والجداول والرياضيات: LTR دائماً

### نشاط الذكاء الاصطناعي
- جميع مربعات التفكير ونشاط الوكيل وشريط الحالة: LTR ثابت (بالإنجليزية)
- يشمل: `.thinking-block`، `.thinking-card`، `.agent-activity-group`، `.composer-status`

## آلية عمل RTL

```
المستخدم يختار العربية ← setLocale('ar') ← 
  ├── RTL_LOCALES.has('ar') = true ←
  ├── localStorage['hermes-rtl-touched'] !== 'true' ←
  ├── document.documentElement.dir = 'rtl' ←
  ├── document.documentElement.classList.add('chat-content-rtl') ←
  └── تطبيق جميع قواعد CSS الخاصة بـ RTL
```

## كيفية استخدام التغييرات

### للتثبيت من الفرع
```bash
cd ~/hermes-webui
git fetch fork
git checkout feature/arabic-language-rtl
```

### بعد التثبيت
1. افتح Hermes WebUI
2. اذهب إلى **Settings → Preferences**
3. اختر **العربية** من Language
4. ستتحول الواجهة كاملة إلى العربية مع RTL

```bash
cd ~/hermes-webui
git fetch upstream
git merge upstream/master
```

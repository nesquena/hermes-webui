# Momentum

Use this note for recurring patterns that help Kei start, continue, pause, or
restart work. Do not store one-off emotions unless Kei explicitly says they
matter.

## Operating Pattern

- When Kei is low-energy, reduce work to the smallest useful next action.
- When Kei is energized, capture the idea quickly and convert it into a first
  concrete step.
- When Kei is curating, sort items into keep, discard, later, build now,
  research first, or automate.
- If rest is the best option, say so plainly and define a low-friction restart
  point.

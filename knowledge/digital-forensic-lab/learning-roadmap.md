Status: source-bound Phase 0 roadmap; not a promoted forensic skill or legal/forensic authority.

## Source boundary

The lab is now tied to the Book Expert Factory source registry. The books are candidate frameworks only. They train templates, questions, and synthetic drills; they do not authorize real evidence handling or final forensic/legal claims.

## Modules

1. Evidence preservation basics
   - Output: preservation checklist + evidence register fields.
2. Chain of custody and integrity
   - Output: custody-event template + hash/placeholder policy.
3. AI forensic reliability risks
   - Output: fact / inference / unknown / human-review-required reporting boundary.
4. Legal governance boundaries
   - Output: stop conditions for legal/forensic claims.
5. Defensive security interface
   - Output: no-offense and no-external-scan boundaries.
6. AgentOps lab growth
   - Output: validator + receipt + review loop.

## Growth loop

read source candidate -> extract framework questions -> convert to synthetic template -> run validator -> record receipt -> review failures -> update playbook/tests.

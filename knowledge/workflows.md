# Workflows

Store recurring workflows here before they are mature enough to become skills.

## Workflow Lifecycle

1. Capture rough workflow here after it succeeds once.
2. Refine after repeated use or failure.
3. Promote to a skill when it is stable and reusable.
4. Keep this note as the index or changelog for the promoted skill.

## Candidates

- Research briefing
- Code review
- Security review
- Momentum restart
- Knowledge curation
